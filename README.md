## Progress

### Data Loading & EDA
- Loaded insurance dataset
- Performed train-test split to avoid data leakage
- Conducted exploratory data analysis (EDA)

### Key Insights
- Charges are right-skewed
- Smoking status is the most important feature
- Smokers have significantly higher charges
- Age increases charges
- BMI strongly affects smokers
- Sex and region have minimal impact
