# 💰 Medical Insurance Cost Predictor

A Machine Learning web application built using **Streamlit** to predict annual medical insurance costs based on user attributes.

---

## 🚀 Live Features

* Predict insurance cost based on:

  * Age
  * BMI
  * Number of children
  * Sex
  * Smoking status
  * Region
* Clean and interactive UI
* Real-time predictions

---

## 🧠 Machine Learning

* Model: Random Forest Regressor
* Dataset: Insurance dataset
* Target variable: `charges` (annual medical cost)

---

## 📊 Dataset Features

| Feature  | Description           |
| -------- | --------------------- |
| age      | Age of the individual |
| bmi      | Body Mass Index       |
| children | Number of dependents  |
| sex      | Male/Female           |
| smoker   | Smoking status        |
| region   | Residential area      |

---

## ⚙️ Tech Stack

* Python
* Scikit-learn
* Pandas, NumPy
* Streamlit

## 📦 Installation

git clone https://github.com/Revanth-reddy080/Medical-Insurance-Cost-Predictor.git
